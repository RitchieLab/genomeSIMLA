MdrPDT Change Report

Listed below are the changes made to MdrPDT and the versions at which they became available to users. The final number in the version string is the build number, and is automatically updated during compilation on certain systems. This number might increase non-sequentially for released builds (though, they will always increase). This is not an error-also, not all builds will be made available to the public (intermittent build increments are very possible during testing and verification of multiple changes).

Date Software Version Problem Solution
Jun 10, 2010 2.0.1.20 Program Segfaults when genotpes are improperly encoded Added a check to verify each allele's encoding and reports correct format before exiting.
Jun 10, 2010 2.0.1.20 Unable to compile MdrPDT for parallelel systems Updated make system to provide parallel capability
Jun 10, 2010 2.0.1.20 Manual incorrectly references CMake Updated manual to reflect current build system
